# TalkMore — Real-Time Chat Application

A real-time chat application with WebSocket-based messaging, user authentication, and live online status.

---

## ✨ Features

- Real-time messaging using WebSockets (Socket.io)
- User authentication with JWT
- Online/offline status indicators
- Chat history persistence with MongoDB
- Responsive UI built with React and Tailwind CSS

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js, Tailwind CSS |
| Backend | Node.js, Express.js |
| Real-time | Socket.io (WebSockets) |
| Database | MongoDB |
| Auth | JWT |

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Mahato-Rambabu/TalkMore-Chat-Application.git
cd TalkMore-Chat-Application

# Install dependencies
npm install

# Add your .env file
# MONGO_URI, JWT_SECRET, PORT

# Run
npm run dev
```
